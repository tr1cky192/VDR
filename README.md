Даний проєкт представляє собою інтегроване віртуальне та доповнене середовище, призначене для високоточного аналізу траєкторій руху промислових роботів.

Скріншот
В директорії StreamingAssets

Вимоги
- Meta Quest 3 (прошивка v62+)
- Unity 2022.3 LTS + Meta XR SDK v65+
- Wi-Fi 6 (для мережевого режиму)

 Структура проєкту
```
Assets/Scripts/   — C#-компоненти
Assets/Scenes/    — Unity-сцени
StreamingAssets/  — набори даних (CSV, GLTF)
docs/             — звіт PDF, презентація PDF
releases/         — підписаний APK
```

Результати
В файлі csv
